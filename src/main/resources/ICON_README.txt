Icon Placeholder
================

Please replace 'app-icon.png' with your actual application icon.

Recommended formats:
- PNG format
- 512x512 pixels or larger
- Transparent background (optional)
- Simple, recognizable design

For macOS .app bundles, the icon will be converted to .icns format automatically by jpackage.
For Windows, it will be converted to .ico format.

You can use online tools or design software to create your icon.
